Welcome to MF-DAT

MF-DAT is an easy to use control panel for data analysis used to analyze performance data from storage systems. It features an assortment of data lenses, graphs, and statistical tools. This tool allows the user to view both individual and aggregated data across individual systems, systems owned by the same company, systems grouped by specific sets of characteristics, and systems overall. We have packaged this whole system into a performant, readily-deployable web application.

This application is a prototype implementation of MF-DAT. It features a sunburst visualization for health scores of many systems at a singular point, and a line graph visualization for health scores of many systems over time. It includes a number of basic data filters, and demonstrates that additional filters would be easy to add. Visualizations are powered by D3.js, and our backend is written in Python using Flask to serve our RESTful API.
